# React Agile Track System

A responsive Agile project tracking web application built using React.js, REST APIs, and Bootstrap.

## Overview
This project was developed as part of my React Developer Trainee Program at Wipro (Jan – Mar 2025).  
It helps Agile teams plan, manage, and track their work efficiently through:
- Role-based authentication (Admin & Employee)
- Task creation, tracking, and completion status
- Team & sprint management
- Analytics dashboard using Excel / Power BI

## Tech Stack
- Frontend: React.js, Bootstrap, JavaScript
- Backend (API Integration): Spring Boot + REST APIs
- Database: MySQL
- Tools: Git, Postman, Power BI
